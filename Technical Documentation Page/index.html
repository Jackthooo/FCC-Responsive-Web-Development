Getting Started

You will need...

  • A computer
  • A code editor
  • A window of the page to check for changes
    • HTML elements

    Elements are the building blocks of HTML. You use the individual parts to build something larger.

    An element is denoted by and opening and closing tag ( < > , < / > ), where the element name sits in between the brackets. See common elements below.

    < header > Place text here < /header > < img src="img url" >
  • some elements are self closing... more info on this later
    • CSS selectors

    CSS selectors are statements that reference html elements.

    They are indicated in a css file by ".element", followed by curly brackets {}, where you place properties and values for the specified HTML element.

    .nav{color: blue; height: 200px; width: 120px;}
  • Above is an example of a declared CSS selector targeting a "nav" element. Some more common properties are:
  • display:
  • font-weight:
    • CSS colors

    There are many ways to represent color in CSS, but for begginers, you really only need to know about the hex format and color names.

    Hex codes are 6 character codes following a #, to indicate properties of the color.

    you can use google to pick hex color codes

    you can also just name the color

    main{ color: red or #FF0000; }
    Images

    You can declare images in HTML with the < img > tag, and providing a src url or the path to a photo in your directory.

    You can further adjust the photo in the CSS file by using the img selector.

    < img src="photo.png">
